# Lab 1

## Object Relational Mapping (ORM)

ORM (Object Relational Mapping) is a technique that connects C# classes with database tables. It allows developers to perform database operations using objects instead of writing SQL statements directly.

### Benefits

- Reduces the amount of SQL code.
- Makes applications easier to maintain.
- Improves development speed.
- Lets developers focus on application logic.

## EF Core vs EF Framework

| Feature | EF Core | EF Framework (EF6) |
|---------|---------|--------------------|
| Platform | Runs on Windows, Linux and macOS | Runs only on Windows |
| Performance | Faster and lightweight | Comparatively slower and heavier |
| Query Support | Supports LINQ, async and compiled queries | Supports LINQ and async queries |
| Development | Continuously updated | Mainly maintenance updates |
