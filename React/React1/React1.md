1. What is SPA (Single Page Application)?

A Single Page Application (SPA) is a web application that loads a single HTML page and updates the content dynamically without reloading the entire page.

Benefits
Faster page navigation
Better user experience
Reduced server requests
Smooth and responsive interface
2. What is React?

React is an open-source JavaScript library developed by Facebook for building fast and interactive user interfaces using reusable components.

How React Works
Divides the UI into reusable components.
Uses a Virtual DOM to detect changes.
Updates only the modified parts of the webpage, improving performance.
3. Difference Between SPA and MPA
SPA	MPA
Loads a single page	Loads a new page for each request
Faster navigation	Slower due to page reloads
Better user experience	Traditional navigation
Uses JavaScript to update content	Server renders each page
4. Pros and Cons of SPA
Pros
Fast and responsive
Better user experience
Less server load
Easy to build interactive applications
Cons
Initial loading may take longer
SEO can be more challenging
JavaScript dependency
5. What is Virtual DOM?

The Virtual DOM is a lightweight copy of the real DOM. React compares the Virtual DOM with the previous version and updates only the changed elements, making applications faster and more efficient.
